AI / ML / GenAI Tools – Brief Definitions

Model & Inference Tools

OpenAI API: Cloud API providing access to large language models for text, vision, embeddings, and tool calling.

Ollama: Local model runner that allows running LLMs on personal machines with simple CLI and API support.

LM Studio: Desktop application for running and testing local LLMs with a GUI.

Hugging Face Transformers: Library providing pre-trained models and tools for NLP, vision, and multimodal tasks.

vLLM: High-throughput LLM inference engine optimized for serving models efficiently.

Triton Inference Server: NVIDIA framework for scalable deployment of ML models on GPUs.

Embeddings & Vector Databases

FAISS: Library for efficient similarity search over high-dimensional vectors.

Pinecone: Fully managed vector database for large-scale semantic search.

Weaviate: Open-source vector database with hybrid search capabilities.

Chroma: Lightweight vector database optimized for local and small-scale RAG systems.

Milvus: Open-source vector database designed for large-scale embedding storage.

RAG & LLM Orchestration Frameworks

LangChain: Framework for building LLM applications using chains, tools, and agents.

LlamaIndex: Data framework focused on connecting LLMs with structured and unstructured data sources.

Haystack: Framework for building search and RAG pipelines with modular components.

Agent & Workflow Tools

AutoGen: Framework for building multi-agent conversational systems.

CrewAI: Agent orchestration framework emphasizing role-based agent collaboration.

Semantic Kernel: Microsoft framework for integrating LLMs with plugins and memory.

Model Training & Fine-Tuning

PyTorch: Deep learning framework for building and training neural networks.

TensorFlow: End-to-end ML framework for training and deployment.

PEFT: Library for parameter-efficient fine-tuning techniques such as LoRA.

LoRA: Fine-tuning method that updates low-rank matrices instead of full model weights.

MLOps & Monitoring

MLflow: Platform for experiment tracking, model versioning, and deployment.

Weights & Biases: Tool for experiment tracking, visualization, and model monitoring.

Prometheus: Monitoring system for metrics collection and alerting.

Grafana: Visualization platform for monitoring dashboards.

Backend & Serving

FastAPI: High-performance Python framework for building ML and AI APIs.

Docker: Containerization tool for packaging applications and dependencies.

Kubernetes: Container orchestration system for scalable deployments.

NGINX: Web server and reverse proxy often used in AI system deployments.

Data Processing

Pandas: Python library for data manipulation and analysis.

NumPy: Core numerical computing library in Python.

Apache Kafka: Distributed event streaming platform for real-time data pipelines.

Apache Airflow: Workflow orchestration tool for batch data pipelines.

Evaluation & Safety

Ragas: Evaluation framework for RAG systems focusing on faithfulness and relevance.

Guardrails AI: Tool for enforcing structured output and safety constraints on LLMs.

Pydantic: Data validation library commonly used to enforce structured LLM outputs.

Visualization & UI

Streamlit: Python framework for building interactive ML and AI dashboards.

Gradio: Tool for creating simple web interfaces for ML models.
